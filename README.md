# invoice-for-all

**Multilingual & Accessible Taiwan Receipt Lottery App**

A mobile-first web prototype that helps everyone — including foreigners and elderly users — participate in Taiwan's government receipt lottery system.

## 🌟 Features

- 🌍 **Multilingual** — zh-TW / EN (日本語、한국어、Español coming soon)
- 👴 **Accessibility first** — font size adjustment during onboarding
- 🤖 **Chatbot onboarding** — step-by-step guided setup with Lucky Angel mascot
- 📱 **Barcode wallet** — display receipt barcode for scanning at checkout
- 🎰 **Prize checking** — auto-check all receipts when lottery results are announced
- 📊 **Prize odds tracker** — shows how to boost your winning chances
